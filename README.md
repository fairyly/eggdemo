# eggdemo
eggdemo

* github: https://github.com/eggjs/egg/
* website: http://eggjs.org/zh-cn/
* doc: http://eggjs.org/zh-cn/intro/quickstart.html


### install

```
$ npm i egg-init -g
$ egg-init egg-example --type=simple
$ cd egg-example
$ npm i

此时目录结构如下：

egg-example
├── app
│   ├── controller
│   │   └── home.js
│   └── router.js
├── config
│   └── config.default.js
└── package.json

启动项目:

$ npm run dev
$ open localhost:7001
```
