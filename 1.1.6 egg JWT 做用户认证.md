# egg-jwt 用户认证

>用户认证主要分为两个部分：  
- 用户通过用户名和密码登录生成并且获取 Token；  
- 用户通过 Token 验证用户身份获取相关信息。  

## install
```
npm i egg-jwt --save
```

## 配置

```
#config/plugin.js

exports.jwt = {
  enable: true,
  package: "egg-jwt"
};


#config/config.default.js

exports.jwt = {
  secret: "123456"
};
```


## 创建一个 token

```
const token = app.jwt.sign({ foo: 'bar' }, app.config.jwt.secret);
```
