# egg 部署到 heroku


## 1.在根目录下新建 Procfile 文件，添加如下内容：

```
web: egg-scripts start
```


## 2.

```
git init
heroku git:remote -a 你的应用名称
git add .
git commit -am "init"
git push heroku master
```


## 3.数据库

mongodb 数据库申请 MLab

修改配置的 mongodb 数据库连接

```
config.mongoose = {
    url: 'mongodb://myblog:myblog@ds139327.mlab.com:39327/apiEgg',
    options: {},
  };
```


## 使用：

```
heroku open
```

打开应用主页。
